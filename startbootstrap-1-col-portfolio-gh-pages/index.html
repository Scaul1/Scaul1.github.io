Sydney Caulfeild Portfolio

My name is Sydney Caulfeild and I am currently a Grade 12 Student at A.Y. Jackson Secondary School. This is my second year of studying computer science. Below are descriptions of the top 10 projects that I would like to be evaluated (there are 9, the 10th project is this GitHub profile).

1. Shiffman Coding Challenge

This project is a replication of Dan Shiffman's Flappy Bird Coding Challenge. He created a YouTube video where he codes a challenge live. I coded along with him to learn the code. I then added to his game by creating a bird image, modifying the gravity and lift variables, and finally by increasing the speed of the pipes as they approach the bird to make it more challenging. This video demonstrates how the game works: the user presses the space bar to make the bird fly and is attempting to avoid the pipes. If they hit a pipe, the pipes turn red to indicate they've been hit. You can view my code for the project by clicking on "View Project".
