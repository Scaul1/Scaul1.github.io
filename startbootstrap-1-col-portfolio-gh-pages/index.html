My name is Sydney Caulfeild and I am currently a Grade 12 Student at A.Y. Jackson Secondary School. This is my second year of studying computer science. I can code in two languages; last year we studied C++ and this year we are coding in JavaScript. I have also learned to code HTML and markdown for my ReadMe files on GitHub Repositores. Here are some of the projects I have been working on this year:

Shiffman Coding Challenge

This project is a replication of Dan Shiffman's Flappy Bird Coding Challenge. He created a YouTube video where he codes a challenge live. I coded along with him to learn the code. I then added to his game by creating a bird image, modifying the gravity and lift variables, and finally by increasing the speed of the pipes as they approach the bird to make it more challenging. This video demonstrates how the game works: the user presses the space bar to make the bird fly and is attempting to avoid the pipes. If they hit a pipe, the pipes turn red to indicate they've been hit. You can view my code for the project by clicking on "View Project".

20-Time Project

Over the course of this semester I have been working on a 20-Time Project, or in other words a project that I dedicate 20% of my time to. I chose to create a soccer injury website that provides information on how to prevent and recover from various injuries. Thsi is a subject that I am very passionate about because I play a lot of soccer and have had a lot of injuries myself. I wanted to share my knowledge on how to recover with others so that they can stay in the game. I used a website called WIX to create my webpage from scratch. I also used JavaScript to code a calendar that I plan on incorporating to demonstrate the different recovery timelines. This image shows the main page for injury prevention.
